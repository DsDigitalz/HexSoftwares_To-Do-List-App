# To-Do List App

A simple, clean, and responsive **dark-themed To-Do List** built using **HTML, CSS, and JavaScript**.
Users can add tasks, edit them, mark them as completed, delete them individually, clear completed tasks, and clear all tasks.
All tasks are stored in the browser using **LocalStorage**, so they stay even after refreshing the page.

---

## Features

### Add Tasks

Users can type a task and press **Add** or hit Enter.

###  Edit Tasks

Each task has an **Edit** button that allows updating the text.

###  Delete Individual Tasks

Each task includes a **Delete** button for easy removal.

###  Mark Tasks as Completed

A checkbox allows toggling completed state. Completed tasks show a line-through style.

###  Clear Completed

Removes only tasks that have been marked as completed.

###  Clear All

Deletes *all* tasks at once (with confirmation).

###  LocalStorage Persistence

Tasks are saved using localStorage, ensuring they remain after page reloads or browser restarts.

###  Fully Responsive & Dark Themed

Modern UI styled with custom CSS (no frameworks).



##  Technologies Used

* HTML5(semantic structure)
* CSS3(dark theme, custom buttons, layout)
* JavaScript(DOM manipulation, storage, event handling)
* LocalStorage for saving tasks
